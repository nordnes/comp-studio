## Cross-service synthesis — strongest convergent patterns

1. **Preview-before-commit + draft→publish is THE editing grammar** for
   consequential numbers (Pave Band Editor, Capboard Apply, Complete draft
   levels, Carta TC accept-with-preview). Validates and sharpens WS-C.
2. **Guardrails = hard stop (block) vs soft stop (justify into the audit trail)**
   with admin-authored explanation copy (Pave Employee Groups, Comprehensive
   caps/floors, Compa auto-approve-in-policy/flag-exceptions). The generosity
   guardrails should adopt this two-tier grammar (WS-G).
3. **Provenance is now a productized UI layer**: Pave consistency labels (sample
   + distribution + margin), Compa citations + verified/preview labeling +
   biweekly refresh dates, Levels.fyi 5-tier verification + row-level recency,
   Aeqium source-tagged imports + band-history audit. COM-182's text flag is a
   generation behind (9a).
4. **Tier/band placement = people-on-band strips**, not tables: Complete's band
   strip with avatars + mid-zone shading + ladder rail with headcounts;
   Ravio's market reference line; compa-ratio + staleness stamp on the person
   card (9b).
5. **Vesting visualization split**: "Today"-marker line w/ vested/unlocked/
   withheld/net popover (Coinbase TM) per-grant; hover-to-next-unlock (Hedgey);
   stepped-bar schedule (Levels.fyi) for static display. Vested ≠ unlocked as
   two co-plotted layers is universal in token tools (9e, WS-F).
6. **Recipient surfaces are separate, simpler, and previewable**: preview-as-role
   chips (Complete Manager/Admin), preview-as-stakeholder (Ledgy/AngelList),
   per-component × per-audience disclosure (Aeqium). The Proposition view
   should get "view as advisor" (WS-G).
7. **Sealed/locked artifacts**: Ledgy locked report history, Hedgey immutable
   lockups + Merkle-root verification, PandaDoc sealed renders. Mutability
   semantics are themselves a trust signal — tag what's binding vs revisable
   (9c).
8. **Inspectability ("show the formula") stays an open gap** across the entire
   field — Capboard's payout math is a black box, Carta's opacity is its top
   complaint, Compa/Pave answer with citations/confidence rather than formulas.
   9d remains a genuine differentiator.
9. **The niche is uncontested**: no service combines dual-instrument
   net-of-strike packages + governance-gated artifacts + scenario sets. Several
   adjacent moves (Compa's agents, Pave's Paige, Aeqium's Analyst) point at AI
   explanation layers as the next competitive axis.
